# <a name="creating-mobile-apps-with-xamarinforms-book"></a>Vytváření mobilních aplikací s Xamarin.Forms knihy

<p><img src="Images/Cover-sml.png" title="Vytváření mobilních aplikací s Xamarin.Forms knihy" align="left" />Knihu <i>vytváření mobilních aplikací pomocí Xamarin.Forms</i> podle Charles Petzold je Průvodce pro učit, jak psát aplikace Xamarin.Forms. Jediným předpokladem je znalost programovacího jazyka C#. Knihu poskytuje rozsáhlou zkoumání do uživatelského rozhraní Xamarin.Forms a věnuje se také animace, MVVM, aktivační události, chování, vlastní rozložení, vlastní renderery a spoustu dalších věcí. 

Knihy publikoval udělali na jaře roku 2016 a od té doby nebyla aktualizována. Existuje hodně knihu této hodnotným zůstane, ale některé [materiál je zastaralá](#ways-in-which-the-book-is-outdated), a některá témata jsou už úplně správná nebo dokončení.

## <a name="ways-in-which-the-book-is-outdated"></a>Způsoby, ve kterém je zastaralá knihy

Od publikování <i>vytváření mobilních aplikací pomocí Xamarin.Forms</i>, přidali několik nových funkcí do Xamarin.Forms. Tyto nové funkce jsou popsány v jednotlivých článcích [Xamarin.Forms](/xamarin/) dokumentaci. 

Další změny způsobily obsah adresáře bude zastaralé:

### <a name="net-standard-20-libraries-have-replaced-portable-class-libraries"></a>Knihovny .NET standard 2.0 nahradil přenosné knihovny tříd

Aplikace Xamarin.Forms obecně používá knihovnu ke sdílení kódu mezi různými platformami. To byl původně přenosnou knihovnou tříd (PCL). Existuje mnoho odkazů na PCLs v celé knihy a sumáře kapitol. 

Knihovny přenosných tříd bylo nahrazeno tématem knihovny .NET Standard 2.0, jak je popsáno v článku [.NET Standard 2.0 Support v Xamarin.Forms](~/xamarin-forms/internals/net-standard.md). Všechny [ukázkový kód](https://github.com/xamarin/xamarin-forms-book-samples) z knihy byl aktualizován na použití knihoven .NET Standard 2.0.

Většinu informací v knize týkajících se role přenosné knihovny tříd zůstává stejná pro knihovny .NET Standard 2.0. Jedním rozdílem je, že má pouze PCL číselnou "profil." Navíc existují některé výhody na knihovny .NET Standard 2.0. Například 20 kapitoly [Async a vstup a výstup souborů](summaries/chapter20.md) popisuje způsob použití odpovídajících platforem pro provádění vstupně-výstupní operace souboru. To již není nezbytné. Knihovna .NET Standard 2.0 podporuje známé [System.IO](xref:System.IO) třídy pro všechny platformy Xamarin.Forms.

Knihovna .NET Standard 2.0 také umožňuje aplikace Xamarin.Forms, aby používaly [ `HttpClient` ](xref:System.Net.Http.HttpClient) při přístupu k souborům přes Internet místo [ `WebRequest` ](xref:System.Net.WebRequest) nebo jiné třídy.

### <a name="the-role-of-xaml-has-been-elevated"></a>Zvýšenými role XAML

<i>Vytváření mobilních aplikací pomocí Xamarin.Forms</i> začíná popisující, jak psát aplikace Xamarin.Forms pomocí jazyka C#. Rozšiřitelné aplikace Markup Language (XAML) není zaveden do [kapitola 7. XAML vs. Kód](summaries/chapter07.md). 

XAML nyní má mnohem větší roli v Xamarin.Forms. Šablony řešení Xamarin.Forms, sadou Visual Studio vytvořit založené na XAML stránkovací soubory. Vývojář pomocí Xamarin.Forms byste se měli seznámit XAML co nejdříve. [EXtensible Application Markup Language (XAML)](~/xamarin-forms/xaml/index.md) část dokumentace Xamarin.Forms obsahuje několik článků o XAML, které vám pomůžou začít.

### <a name="supported-platforms"></a>Podporované platformy 

Xamarin.Forms už podporuje Windows 8.1 a Windows Phone 8.1. 

Knihy někdy díky odkazy _modulu Windows Runtime_. Toto je termín, který zahrnuje rozhraní API Windows používá v několika verzích Windows a Windows Phone. Novější verze Xamarin.Forms omezuje podporuje univerzální platformu Windows, což je rozhraní API pro Windows 10 a Windows 10 Mobile.

Knihovna .NET Standard 2.0 nepodporuje všechny verze Windows 10 Mobile. Aplikace Xamarin.Forms pomocí knihovny .NET Standard proto nebude spouštět na zařízení s Windows 10 Mobile. Aplikace Xamarin.Forms dál běží na Windows 10 desktop, verze 10.0.16299.0 a vyšší. 

Xamarin.Forms nabízí podporu verze preview pro [Mac](~/xamarin-forms/platform/mac.md), [WPF](~/xamarin-forms/platform/wpf.md), [GTK #](~/xamarin-forms/platform/gtk.md), a [Tizen](/xamarin-forms/platform/tizen.md) platformy.
