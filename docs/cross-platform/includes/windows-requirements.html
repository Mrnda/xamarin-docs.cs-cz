Doporučeno Poznámky
Operační systém Windows 10 Minimální verzi operačního systému je systém Windows 7. Podpora prostředí Xamarin.Forms Windows vyžaduje Windows 8.1 a podpora Xamarin.Forms UWP vyžaduje Windows 10.
Xamarin.iOS iOS 10 SDK
na Macu nainstalované		 K sestavení projektů iOS v systému Windows vyžaduje:
  • Visual Studio 2015 nebo novější, a
  • do počítače Mac, síťově přístupné z počítači s Windows, který splňuje minimální požadavky na spuštění Xamarin v systému macOS (výše).
Xamarin.Android Android 6.0 / na úrovni rozhraní API 23 Při použití na nejnovější SDK, stále můžete vybrat Android starší verze, nebo můžete vytvořit pro starší verze sady SDK v případě potřeby.
Xamarin.Forms Xamarin.Forms aplikace pro iOS a Android můžete nasadit na příslušné platformy, jak je popsáno výše.
Pomocí sady Visual Studio také znamená, že můžete testovat aplikace pro Windows a univerzální platformu Windows (ve Windows 10) s použitím emulátorů společnosti Microsoft. Aplikace pro Windows můžete otestovat přímo na vývojovém počítači.
Xamarin.Mac Projekty Xamarin.Mac (aplikace klasické pracovní plochy systému macOS) lze otevřít v sadě Visual Studio a zkompilovat ke kontrole chyb, ale aplikací systému Mac nemůže aktuálně být vytvořena pro distribuci v sadě Visual Studio. Najdete v poznámkách k verzi na Xamarin.Mac podporu Další informace o omezeních Mac projekty v sadě Visual Studio.
